miHoYo Tool
专治各种“禁止粘贴”的输入框。

有些网页（如网银）、远程桌面或软件禁止使用 Ctrl+V 粘贴密码或文本。本工具可以读取你剪贴板里的内容，模拟键盘通过“打字”的方式帮你输进去。

🚀 怎么用
下载：下载 miHoYo Tool.exe。

运行：右键选择 “以管理员身份运行”（重要！否则可能无法输入）。

复制：正常复制你要输入的文字（Ctrl+C）。

输入：点一下你要输入的框，按下 F9 键。

完成：软件会自动把字打进去。

⌨️ 快捷键
F9：开始输入（默认）

F10：停止输入（默认）

你也可以在软件界面上修改这两个按键。

⚙️ 常见问题
输入太慢/太快？ 在软件界面调整“基础延迟”即可。一般保持默认就行。

为什么没反应？ 请确保你是以管理员身份运行的本软件。
